Установить сегмент данных, где хранятся сообщения и буфер для строки.
Вывести сообщение “Введите строку:” на экран с помощью функции DOS 09h.
Ввести строку с клавиатуры с помощью функции DOS 0Ah. Строка записывается в буфер, где первый байт - максимальная длина буфера, второй байт - фактическая длина строки, а остальные байты - символы строки.
Получить длину строки из второго байта буфера и сохранить ее в переменной len.
Проверить, что длина строки не меньше 9 символов. Если меньше, то завершить программу.
Обменять местами третий и девятый символы строки с помощью команды xchg.
Вычислить значение S4 - (S5 - S6) с помощью команд sub и add и записать его во второй символ строки.
Вывести сообщение “Результат:” на экран с помощью функции DOS 09h.
Ожидать нажатия клавиши с помощью функции BIOS 00h.
Завершить программу с помощью функции DOS 4Ch.
